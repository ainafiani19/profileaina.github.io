Profil Singkat

Saya Aina Nurfiani, lulusan S1 Perencanaan Wilayah dan Kota dari Universitas Brawijaya. Saya memiliki pengalaman dalam analisis spasial, pemetaan, dan pengolahan data geospasial menggunakan perangkat lunak seperti ArcGIS, QGIS, AutoCAD, SketchUp, dan Microsoft Office.

Portofolio ini menampilkan beberapa pengalaman kerja dan proyek studi yang berkaitan dengan analisis wilayah, pemetaan, serta penyusunan dokumen perencanaan. Melalui pengalaman tersebut, saya siap mendukung pekerjaan yang membutuhkan kemampuan analisis spasial dan pengolahan data geospasial.

Pengalaman Kerja

KLHS WP Cicalengka

Magang | Dokumen KLHS

Terlibat dalam analisis spasial, overlay KRP berdampak, analisis DPSIR, dan penyusunan rekomendasi.

  • Menyusun sub-bab analisis pengaruh lingkungan strategis yang mencakup lebih dari 7 jenis analisis dalam dokumen KLHS.
  • Melakukan analisis DPSIR (Driver–Pressure–State–Impact–Response) untuk mengidentifikasi hubungan antara aktivitas pembangunan dan dampak lingkungan.
  • Merancang dan menyusun alternatif rekomendasi kebijakan sebagai bagian dari penyusunan dokumen KLHS.
  • Menyusun lebih dari 50 slide presentasi untuk mendukung kegiatan Konsultasi Publik KLHS bersama pemangku kepentingan.

Project Studi

Studio Perencanaan Wilayah

Studio ini berfokus pada penyusunan rencana strategis wilayah berbasis sektor yang kemudian diintegrasikan untuk menghasilkan konsep pengembangan kawasan yang komprehensif. Dalam proyek ini saya melaksanakan lebih dari 150 jam survei lapangan, termasuk wawancara dengan penduduk lokal, pengelola pariwisata, pemilik usaha, dan instansi pemerintah. Berdasarkan data tersebut, saya melakukan 5+ jenis analisis untuk menyusun konsep pengembangan kawasan serta berperan dalam penyusunan dan presentasi laporan akhir di hadapan dosen pembimbing dan pemangku kepentingan lokal.

Studio Perancangan Kota

Pada Studio Perancangan Kota, saya terlibat dalam penyusunan dokumen RTBL mulai dari survei lapangan hingga perancangan kawasan. Saya melaksanakan lebih dari 160 jam survei lapangan yang difokuskan pada analisis lalu lintas dan karakteristik fisik bangunan. Berdasarkan hasil survei tersebut, saya menyusun lebih dari lima peta tematik dan analisis spasial untuk mendukung perencanaan kawasan exit tol serta terlibat dalam perancangan landmark, streetscape koridor jalan, dan desain pedestrian menggunakan AutoCAD dan SketchUp.

Studio Perencanaan Permukiman Kota

Pada studio ini saya mempelajari proses survei lapangan, pengolahan data, serta penyusunan laporan perencanaan secara sistematis. Saya bertanggung jawab mengelola master data menggunakan Excel dan Word, memanfaatkan rumus seperti SUMIF dan Pivot Table untuk pengolahan data, serta terlibat dalam pembuatan peta dan analisis spasial menggunakan ArcGIS.
