Profil Singkat

Saya Aina Nurfiani, lulusan S1 Perencanaan Wilayah dan Kota dari Universitas Brawijaya. Saya memiliki pengalaman dalam analisis spasial, pemetaan, dan pengolahan data geospasial menggunakan perangkat lunak seperti ArcGIS, QGIS, AutoCAD, SketchUp, dan Microsoft Office.

Portofolio ini menampilkan beberapa pengalaman kerja dan proyek studi yang berkaitan dengan analisis wilayah, pemetaan, serta penyusunan dokumen perencanaan. Melalui pengalaman tersebut, saya siap mendukung pekerjaan yang membutuhkan kemampuan analisis spasial dan pengolahan data geospasial.
